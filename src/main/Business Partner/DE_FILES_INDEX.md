## 💡 YOUR FRAMEWORK (Memorize This!)

Use this for EVERY question:

```
CLARIFY
├─ Ask 3-4 clarifying questions first
├─ Don't assume you understand the problem
└─ Show you care about solving the right problem

DATA PROBLEM
├─ What data sources?
├─ What schema/modeling?
├─ What latency/quality/scale?
└─ Identify data-specific challenges

SOLUTION
├─ Propose pragmatic approach
├─ Discuss trade-offs
├─ Show how you'd handle quality
└─ Explain testing/monitoring

COLLABORATION
├─ Show you'd work WITH teams
├─ Explain communication plan
├─ Discuss how you'd iterate
└─ Show flexibility

IMPACT
├─ Connect to business outcome
├─ Explain how success is measured
├─ Show you understand importance
└─ Ask for feedback
```
